# Null-Interferometry
A simulation tool for nulling interferometry that models various interferometer configurations and telescope parameters. It analyzes the impact of instrumental errors, pointing jitter, and stellar activity on exoplanet detection SNR, helping to evaluate and optimize high-contrast imaging performance.
Nulling Interferometry Simulation Tool
Overview

This project provides a simulation framework for nulling interferometry, designed to study and optimize the direct imaging of exoplanets.
The tool enables users to simulate different interferometer configurations and telescope parameters, and to analyze how various noise sources affect the signal-to-noise ratio (SNR) in exoplanet detection.

Features

Flexible configuration of telescope parameters (number of telescopes, aperture size, baseline length, and geometry)

Simulation of instrumental, pointing jitter, and stellar activity noise

Calculation of key performance metrics such as null depth, imaging efficiency, and SNR

Modular and extendable design for customized interferometric studies

Applications

This tool is intended for researchers and engineers working on:

High-contrast imaging and interferometer design

Exoplanet detection simulations

Mission performance analysis and optimization

Future Development

Planned updates include:

Multi-wavelength and dynamic stability simulations

Integration with exoplanet population and orbital models

Visualization tools for interferometric performance metrics
